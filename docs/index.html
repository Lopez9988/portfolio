Projects

Intersection Scale Model:

The Scale Model was a 3-person group class project for ECE 102 aimed at simulating a 4-way intersection. For this project, we utilized the Labjack microcontroller interfaced with MATLAB. I was personally in charge of handling the software functionality, which consisted of several digital I/O on the controller, inputs on buttons for the pedestrian indicator, and the intersection lights.

Link: Traffic Interection Project Report

ChocAn Data Processing System:

The ChocAn System was a 6-person group project for CS300. The purpose of the project was to create a data-processing system to help individuals with chocolate addiction. We followed proper software engineering proceedures and created requirements, design, testing, and report plans to help throughout the process. I was personally responsible of programming the generation of reports created for the manager and provider users, which ranged from weekly users, costs, and services provided.
